## Description

The **Create Request** menu item type is used to display a form allowing
users to submit an information request (Export or Remove).

## How To Access

To create a new **Create Request** menu item:

- Select **Menus → \[name of the menu\]** from the Administrator
  menu (for example, **Menus → Main Menu**). Then...
  - Select the New button in the Toolbar. Then...
  - Select the Menu Item Type Select button.
  - In the modal dialog select the Privacy item to open a list and then
    select the **Create Request** item.

To edit an existing Create Reques menu item:

- Select its Title in the Menus: Items list.

## Form Fields

- **Menu Title:** The title that will display for this menu item.
- **Alias**. The internal name of the item. Normally, you can leave this
  blank and Joomla will fill in a default value Title in lower case and
  with dashes instead of spaces.
